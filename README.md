形式计算理论课程: 形式语言与自动机  
形式计算的基础是符号化和语言. 形式语言与自动机是研究符号化表达 (即形式语言) 以及符号化识别与处理 (即自动机) 的理论.

不确定数学理论课程: 概率论与数理统计  
- 确定性现象: 在一定条件下必然发生某种结果的现象.
- 偶然性现象: 在一定条件下, 有多种可能结构发生, 事先不能预测.

偶然性现象特征:  
- 随机性 (偶然性)
- 大量实验条件下, 其结果具备规律性.
